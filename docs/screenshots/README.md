# Screenshots Directory

This directory contains feature screenshots for the README.

## Required Screenshots

1. **grid-recon.png** - Grid Recon 3D map showing Texas with satellite imagery
2. **dashboard.png** - Main dashboard with real-time metrics
3. **insights.png** - AI Insights page with recommendations
4. **agents.png** - Agent Orchestrator visualization
5. **governance.png** - AI Governance dashboard
6. **cyber-sim.png** - Cyber simulation interface

## How to Capture

1. Navigate to `http://localhost:3000`
2. Go to each feature page
3. Take a screenshot (Cmd+Shift+4 on Mac, Win+Shift+S on Windows)
4. Save with the filename listed above
5. Place in this directory

## Recommended Resolution

- 1920x1080 or higher
- PNG format for best quality
- Capture the full browser window or just the content area
